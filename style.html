HTML Style Attribute

HTML Stands for "Hypertext Markup Language." HTML is the language used to create webpages.

Introduction of HTML

"Hypertext" refers to the hyperlinks that an HTML page may contain. "Markup language" refers to the way tags are used to define the page layout and elements within the page

Why Style Attribute is used ?

The style attribute specifies an inline style for an element. The style attribu will override any style set globally, e.g. styles specified in the style tag or in an external style sheet.
